# Devre Market Web

Devre Market icin bagimsiz statik web projesi.

## Proje Amaci

Bu proje elektronik malzeme satisi, egitim icerikleri ve proje paketleri icin ilk web vitrinidir.
Mevcut gomulu yazilim projesinden bagimsizdir.

## Klasor Yapisi

- `index.html`: Ana sayfa
- `src/styles/main.css`: Tasarim ve responsive duzen
- `src/scripts/main.js`: Sepet sayaci ve urun filtreleri
- `assets/`: Gorsel, logo ve dokuman varliklari icin ayrildi
- `docs/`: Is plani, icerik ve teknik notlar

## Calistirma

Ek kurulum gerekmez. `index.html` dosyasi tarayicida acilabilir.

## Sonraki Teknik Adimlar

1. Gercek urun listesini JSON veya basit bir panelden beslemek.
2. WhatsApp teklif linkini forma baglamak.
3. Alan adi, hosting ve SSL kurulumu yapmak.
4. Odeme sistemi gerekiyorsa iyzico, PayTR veya benzeri servis entegrasyonu eklemek.
